# ArtisanAI

🎨 Assistant IA pour artisans - Devis, factures et chat

## 🚀 Fonctionnalités

- ✅ Authentification sécurisée (Supabase)
- ✅ Création de devis avec IA
- ✅ Génération de factures PDF
- ✅ Chat assistant IA intégré
- ✅ Gestion des clients
- ✅ Interface adaptée aux artisans

## 🛠️ Technologies

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Base de données**: Supabase
- **IA**: OpenAI API
- **PDF**: pdf-lib

## 📦 Installation

```bash
# Cloner le projet
git clone https://github.com/marcusglad4-sys/artisanai.git
cd artisanai

# Installer les dépendances
npm install

# Configurer les variables d'environnement
cp .env.local.example .env.local
# Remplissez .env.local avec vos clés API

# Lancer le serveur de développement
npm run dev
```

Ouvert sur [http://localhost:3000](http://localhost:3000)
